# Internshield – Internship & Job Verification Platform

## Overview

Internshield is a web-based platform designed to help students and job seekers identify whether an internship or job opportunity is genuine or potentially fake/spam. The platform analyzes company links and provides insights such as red flags, green flags, and an overall trust score to help users make safer career decisions.

## Features

* Internship and job verification system
* Spam/fake opportunity detection
* Trust score analysis
* Red flags and green flags identification
* User-friendly frontend interface
* Backend support for data processing
* Browser extension integration

## Technologies Used

* HTML

* JavaScript
* Backend Technologies
* Batch File Automation

## Purpose

The main goal of this project is to protect students and fresh graduates from fake internship scams and misleading job opportunities by providing an easy-to-use verification platform.
